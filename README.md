# Encrypted Slack Channel for Security Incident Response
### NYU CS6903/4783 Applied Cryptography — Project 2.2

> End-to-end encrypted group messaging over Slack. Slack is treated as an untrusted transport relay — all cryptographic operations happen on sender and recipient machines. Slack never sees plaintext.

---

## Setup

```bash
pip install -r requirements.txt
cp .env.example .env
# Add your Slack Bot Token to .env
```
